Hello and welcome to the preliminary DAMARIS homepage!


DAMRARIS stands for "Darmstadt Magnetic Resonance Imaging Spectrometer"

In first place this project develops a framework for NMR spectrometer controls. A lot of other projects could be found, providing NMR data processing and evaluation software. So this project is focusing on:
  • creating control software for a standard NMR-spectrometer (pulses and data acquisition)
  • providing a toolkit for tailoring spectrometer control softare on your own
  • making the control hardware characteristics transparent to the spectrometer developer
  • be a forum for exchanging experiences with the software and underlying hardware
If you are curious and have questions - please join our mailinglist.

Particularly this page should be a collection of resources:
  • source repository with subversion repository, browsable
  • documentation of backend (browsable, English language)
  • poster presented at XIII. Ampere NMR summer school in Zakopane, Poland, June 2005 (PDF)
  • presentations and other stuff (mainly in German, created with OpenOffice)
Please be aware that we are distributing our own code with the esser GNU public license (LGPL), but in the drivers directory there are several files, that are several vendor specific headers. This code is freely downloadable but not our property, so not subject to the LGPL.

Supported hardware since now:
  • SpinCore PulseBlaster pulse generators with and without DDS (PCI card)
  • Datel PCI-416 data acquisition board (PCI card, no longer buyable)
  • Spectrum MI 40xx series data acquisition boards (PCI cards)
  • TiePie Handyscope 3, data acquisition card (USB device)
  • Eurotherm 2016e temperature control (serial port device)
Used software (dependencies of this project):
  • GNU C++ Compiler (Linux: standard distribution, Windows: Cygwin)
  • Xerces C++ XML support library (Linux: please install also "devel" packages, Windows: Cygwin)
  • doxygen and graphiz for c++-documentation (Linux: standard distribution, Windows: Cygwin)
  • make (Linux: standard distribution, Windows: Cygwin)
